### Keywords

- **Latency** refers to the delay that happens between when a user takes an action on a network or web application and when it reaches its destination, which is measured in milliseconds.
- **Back of the Envelope** Estimation is a quick way to roughly calculate system requirements like traffic, storage, bandwidth, and servers using simple math, so we can plan a scalable and cost-effective system design.
- **DAU/MAU** - Daily Active Users or Monthly Active Users
- **RPS** - Requests Per Second
- **Bandwidth** - The data transfer capacity of a network
- **CAP Theorem** - The CAP theorem says that a distributed system can deliver only two of three desired characteristics: consistency, availability and partition tolerance
